# Tulip

Платформа для управляющей компании в Турции:

- Telegram-бот для клиентов
- личный кабинет клиента
- личный кабинет управляющей компании
- единый backend
- хранение операционных данных в PostgreSQL

## Цель

Собрать MVP, в котором управляющая компания сможет:

- вести объекты, собственников и арендаторов
- принимать заявки от клиентов
- отслеживать оплаты, документы и статусы задач
- общаться с клиентами через Telegram

Клиенты смогут:

- авторизоваться через Telegram
- смотреть информацию по своему объекту
- оставлять заявки
- видеть платежи, документы и статус обращений

## Структура проекта

```text
apps/
  api/               Backend API и бизнес-логика
  bot/               Telegram-бот
  client-cabinet/    Личный кабинет клиента
  admin-cabinet/     Кабинет управляющей компании
packages/
  core/              Общие типы, схемы, доменные модели
docs/                Документация проекта
prototype/           Визуальный интерактивный каркас продукта
```

## Предлагаемый стек

- `TypeScript`
- `Node.js`
- `Fastify` или `NestJS` для API
- `Next.js` для веб-кабинетов
- `Telegram Bot API` через `grammy`
- `PostgreSQL` как основная база данных
- `Adminer` для локального просмотра базы
