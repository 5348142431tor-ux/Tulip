# Admin Cabinet

Автономный frontend-каркас кабинета управляющей компании.

## Что внутри

- дашборд с KPI
- очередь заявок
- список клиентов
- реестр объектов
- контроль платежей
- документы

## Точка входа

- [index.html](./index.html)

## Архитектурная идея

Интерфейс пока работает на sample-данных из `app.js`, но структура уже подготовлена под замену на API:

- `dataStore` хранит сущности
- `views` описывают экраны
- `render*` функции отвечают за визуализацию конкретных модулей

Следующий шаг после этого каркаса:

- подключить backend API
- загрузку данных из PostgreSQL через backend
- авторизацию сотрудников
