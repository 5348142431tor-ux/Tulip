# Client Cabinet

Кабинет клиента для расширенных сценариев:

- просмотр объекта
- документы
- платежи
- история заявок

Контур входа:

- прямого логина по роли больше не должно быть
- клиент приходит только из Telegram-бота
- бот получает для клиента ссылку с hex-кодом доступа к конкретной квартире
- кабинет открывается по маршруту вида `/client-cabinet/access/:accessCode`
- API валидирует hex-код и возвращает только данные этой квартиры
